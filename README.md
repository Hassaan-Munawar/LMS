# 🎓 Complete LMS

A modern, full-stack **Learning Management System (LMS)** built to provide a seamless online learning experience for students, instructors, and administrators. This project focuses on scalability, performance, security, and a clean user interface.

## ✨ Features

### 👨‍🎓 Student

* Secure authentication and authorization
* Browse and search courses
* Enroll in courses
* Watch video lectures
* Track learning progress
* Complete quizzes and assignments
* Download course resources
* Earn course certificates
* Responsive dashboard

### 👨‍🏫 Instructor

* Create and manage courses
* Upload video lectures
* Organize course sections
* Create quizzes and assignments
* Monitor student enrollments
* Track student performance
* Manage course content

### 👨‍💼 Admin

* Manage users
* Manage instructors
* Manage students
* Manage courses
* Review platform analytics
* Approve instructor requests
* Control platform settings

## 🚀 Tech Stack

### Frontend

* React

* Tailwind CSS
* shadcn/ui
* Axios

### Backend

* Node.js
* Express.js

### Database

* MongoDB
* Mongoose

### Authentication

* JWT Authentication
* Role-Based Access Control (RBAC)

### Storage

* Cloudinary (for media uploads)

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/complete-lms.git
```

### Navigate to the project

```bash
cd complete-lms
```

### Install dependencies

Frontend

```bash
cd client
npm install
```

Backend

```bash
cd ../server
npm install
```

## ▶️ Run the Project

Frontend

```bash
npm run dev
```

Backend

```bash
npm run dev
```
